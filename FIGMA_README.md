# Дизайн-система для Figma

Этот файл содержит полное описание дизайн-системы проекта MAX Campus Mini App для использования в Figma.

## Файлы

- `figma-design-system.json` - JSON файл с полным описанием дизайн-системы
- `figma-components-reference.html` - HTML файл с визуализацией компонентов (для референса)

## Как использовать в Figma

### Вариант 1: Ручное создание компонентов

1. Откройте Figma и создайте новый файл
2. Используйте информацию из `figma-design-system.json` для создания:
   - **Color Styles** - создайте стили цветов из раздела `colors`
   - **Text Styles** - создайте текстовые стили из раздела `typography`
   - **Components** - создайте компоненты из раздела `components`

### Вариант 2: Импорт через плагин

1. Установите плагин Figma для импорта JSON (например, "Design Tokens" или "Figma Tokens")
2. Импортируйте `figma-design-system.json` через плагин

### Вариант 3: Использование как референса

1. Откройте `figma-components-reference.html` в браузере для визуального референса
2. Используйте значения из JSON для точного воссоздания компонентов

## Структура дизайн-системы

### Цвета

- **Primary**: `#3C35F2` - основной цвет
- **Accent**: `#2980F2` - акцентный цвет
- **Background**: `#F5F5F5` (основной), `#FFFFFF` (карточки), `#F1F2F4` (элементы)
- **Text**: `#000000` (основной), `#666666` (вторичный), `#999999` (третичный)

### Типографика

- **Font Family**: Системные шрифты (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- **Размеры**: 10px (tiny), 13px (caption), 14px (small), 15px (medium), 16px (large), 18px (subheading), 20px (heading), 22px (title)

### Компоненты

#### Кнопки
- **Primary**: Синий фон (#2980F2), белый текст, скругление 12px
- **Secondary**: Серый фон (#F5F5F5), черный текст, скругление 12px

#### Карточки
- **Основная карточка**: Белый фон, скругление 20px, отступ 18px
- **Карточка занятия**: Серый фон (#F1F2F4), скругление 10px, отступ 12px 16px

#### Навигация
- **Header**: Высота 56px, фон #F5F5F5, sticky позиционирование
- **Bottom Navigation**: Высота 70px, белый фон, фиксированная внизу

### Отступы

Базовые отступы: 4px, 6px, 8px, 12px, 16px, 20px, 24px

### Скругления

- 8px (small)
- 10px (medium)
- 12px (large)
- 16px (xlarge)
- 20px (xxlarge)
- 50% (round)

## Страницы приложения

1. **Главная** (`/`) - расписание и мероприятия
2. **Расписание** (`/schedule`) - недельное расписание
3. **Мероприятия** (`/events`) - список мероприятий
4. **Дедлайны** (`/deadlines`) - список дедлайнов
5. **Преподаватели** (`/teachers`) - список преподавателей
6. **Сервисы** (`/services`) - сетка сервисов
7. **Профиль** (`/profile`) - профиль пользователя
8. **Настройки** (`/settings`) - настройки приложения
9. **Практика** (`/practice`) - поиск практики
10. **Поддержка** (`/support`) - обратная связь

## Рекомендации

1. Создайте Frame размером 375x812px (стандартный iPhone)
2. Используйте Auto Layout для компонентов
3. Создайте Variants для кнопок (primary/secondary, разные размеры)
4. Используйте Constraints для адаптивности
5. Создайте Component Sets для иконок навигации

## Иконки

Все иконки находятся в папке `miniapp/public/`. Используйте их как референс или импортируйте в Figma.
