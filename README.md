# Легкий чат-бот для мессенджера MAX

Простой и функциональный чат-бот, разработанный для мессенджера MAX с использованием официального API клиента.

## Возможности

- ✅ Команды для взаимодействия с пользователями
- ✅ Приветствие при первом запуске
- ✅ Обработка текстовых сообщений
- ✅ Проверка работоспособности (ping-pong)
- ✅ Показ текущего времени
- ✅ Эхо-функция для повторения сообщений
- ✅ Справка по командам

## Установка

1. Убедитесь, что у вас установлен Node.js версии 18.18.0 или выше
   ```bash
   node --version
   ```

2. Установите зависимости:
   ```bash
   npm install
   # или
   yarn install
   ```

3. Настройте переменные окружения:
   
   Создайте файл `.env` в корне проекта или установите переменную окружения:
   ```bash
   BOT_TOKEN=ваш_токен_бота
   ```
   
   Или используйте токен напрямую в коде (для разработки).

## Запуск

### Режим разработки (с автоперезагрузкой)
```bash
npm run dev
# или
yarn dev
```

### Сборка и запуск
```bash
# Сборка TypeScript в JavaScript
npm run build

# Запуск скомпилированного бота
npm start
```

## Доступные команды

- `/start` - Запустить бота и получить приветствие
- `/help` - Показать список доступных команд
- `/ping` - Проверить работоспособность бота (ответит "pong")
- `/time` - Узнать текущее время
- `/echo <текст>` - Повторить ваше сообщение

Для разработки рекомендуется использовать режим разработки с `ts-node`:
```bash
npm run dev
```

При изменении кода, бот автоматически перезапустится (если используется nodemon).
