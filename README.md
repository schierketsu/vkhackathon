# Простой чат-бот для Max

Простой чат-бот, созданный с использованием Max Bot API.

## Установка

1. Установите зависимости:
```bash
npm install
```

или

```bash
yarn install
```

## Запуск

1. Скомпилируйте TypeScript код:
```bash
npm run build
```

2. Запустите бота:
```bash
npm start
```

Или выполните обе команды сразу:
```bash
npm run dev
```

## Функциональность

Бот поддерживает следующие команды:
- `/start` - Начать работу с ботом
- `/help` - Показать список команд
- `/echo` - Активировать эхо-режим

Также бот реагирует на:
- Сообщения с текстом "привет" или "hello"
- Все остальные текстовые сообщения (повторяет их)

## Используемые технологии

- TypeScript
- @maxhub/max-bot-api
