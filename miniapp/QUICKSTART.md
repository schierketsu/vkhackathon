# Быстрый запуск мини-приложения

## Шаг 1: Установка зависимостей

```bash
cd miniapp
npm install
```

## Шаг 2: Запуск API сервера

В первом терминале:

```bash
npm run api
```

API сервер запустится на порту 3001.

## Шаг 3: Запуск приложения

Во втором терминале:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Примечания

1. Мини-приложение использует ту же базу данных, что и бот (`data/campus.db`)
2. Если база данных не существует, она будет создана автоматически
3. Для работы с расписанием необходимо сначала выбрать группу в настройках
4. Функции получения расписания и списка преподавателей требуют дополнительной реализации (помечены TODO в коде)

## Интеграция с реальными функциями

Для полноценной работы необходимо:

1. Реализовать функции `getTodaySchedule`, `getTomorrowSchedule`, `getCurrentWeekSchedule` в `api/shared-utils.ts`, используя логику из `src/utils/timetable.ts`
2. Реализовать функцию получения списка групп в API
3. Добавить интеграцию с MAX Bridge для получения `userId` пользователя
